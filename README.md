# RapidCare Ambulance Service

A responsive, SEO-ready ambulance service landing page built with Next.js and designed for deployment on Vercel.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Import the GitHub repository into Vercel, or run `npx vercel --prod` after logging in. Vercel automatically detects Next.js.
